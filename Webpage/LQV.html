Live Quest Visualization


Welcome to the homepage of our Google add-on!



This is an add-on, that was developed for teachers, who use google forms to create tests for students to assess them.
You, as a creator of the form, get an overview of the responses of students for each form
as a series of graphs in the Forms menu.




The link to the add-on can be found in the Add-on menu,
where you can also find the instructions for using it in your form.




Our Add-on uses Learning Locker Storage, which is a open source data repository to store learning activity.
For storing data, it uses learning activity statements generated by xAPI (Tin Can).
Our Add-on captures Google Forms data using Tin Can and then sends to Learning Locker.
For visualizing we use D3, which is a JavaScript library to manipulate data.
